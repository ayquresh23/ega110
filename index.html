Measurements: Data & Errors

Before any analytical result means anything, you need to understand what errors are, where they come from, and how to describe the quality of a set of measurements.

What is analytical chemistry?

Definition

Analytical chemistry is the science that develops and applies methods, instruments, and strategies to obtain information on the composition and nature of matter.

In practice: you have a sample — soil, water, a tablet, an alloy — and you want to know what's in it and how much. Every technique in this module is a different answer to that question.

Accuracy vs Precision

These two words are used interchangeably in everyday language. In analytical chemistry they mean strictly different things and you cannot use them interchangeably.

Accuracy

How close a measured value is to the true (accepted) value.

Measured by: error (absolute or relative).

A result can be precise but inaccurate — if a systematic error consistently shifts all measurements the same way.

Precision

How close replicate measurements are to one another.

Measured by: standard deviation.

A result can be precise but inaccurate, or accurate but imprecise (scattered around the true value).

🎯
Think of a dartboard. Accurate = darts clustered around the bullseye. Precise = darts clustered tightly together (even if away from the bullseye). You want both — tight cluster on the bullseye.

Types of Error

Random (indeterminate) error

Unpredictable, affects precision. Caused by uncontrollable variables — air currents, vibrations, fluctuating power supply. Leads to scatter around the mean.

Cannot be eliminated — only reduced by taking more measurements and averaging.

Systematic (determinate) error

Consistent, reproducible, affects accuracy. Always shifts results in the same direction by roughly the same amount.

Can in principle be identified and corrected. Sources: instrumental (uncalibrated zero), methodological (wrong procedure), personal (observer bias), environmental.

Gross error

Occasional large mistakes — spilling sample, misreading a number, recording wrong units. Produces outliers. Detected by the Q-test (covered in Topic 2).

Absolute and Relative Error

Absolute error
$$E_i = \lvert x_i - x_t \rvert$$
xᵢmeasured value
xₜtrue (accepted) value
Eᵢabsolute error — same units as measurement
Relative error
$$R_i = \dfrac{\lvert x_i - x_t \rvert}{x_t}$$
Rᵢrelative error — dimensionless (multiply ×100 for %)

Relative error is useful for comparing errors across different measurement scales — 0.2 cm error matters more when measuring a 1 cm object than a 100 cm object.

Worked example — absolute and relative error

An object has true diameter xₜ = 10.0 cm. Three measurements: 9.8, 10.2, 9.9 cm. Calculate absolute errors, average absolute error, relative errors, and average relative error.

E₁ = |9.8 − 10.0| = 0.2 cm; E₂ = |10.2 − 10.0| = 0.2 cm; E₃ = |9.9 − 10.0| = 0.1 cm
Average absolute error = (0.2 + 0.2 + 0.1)/3 = 0.167 cm
R₁ = 0.2/10.0 = 0.020; R₂ = 0.020; R₃ = 0.1/10.0 = 0.010
Average relative error = (0.020 + 0.020 + 0.010)/3 = 0.0167 (1.67%)

Statistical Descriptors

Sample mean
$$\bar{x} = \dfrac{\sum x_i}{N}$$
Sample standard deviation (Bessel's correction)
$$s = \sqrt{\dfrac{\sum(x_i - \bar{x})^2}{N-1}}$$
N−1degrees of freedom — NOT N. Using N−1 gives an unbiased estimate of σ.
Standard error of the mean & RSD
$$S_m = \dfrac{s}{\sqrt{N}} \qquad \text{RSD} = \dfrac{s}{\bar{x}}\times 100\%$$
Population (σ, N) vs Sample (s, N−1)

When you have the entire population, use σ and divide by N. When you have a sample (which is almost always the case), use s and divide by N−1. In exams, use s unless explicitly told σ is known.

The Gaussian (Normal) Distribution

When only random errors are present, the results form a Gaussian (bell-shaped) distribution centred on the true mean μ.

68-95-99.7 rule
  • ~68% of measurements fall within ±1σ of the mean
  • ~95% fall within ±2σ
  • ~99.7% fall within ±3σ

Increasing σ → wider, flatter curve. Decreasing σ → taller, narrower curve. Changing μ → shifts the curve sideways without changing its shape.

Confidence Intervals — σ known (z-statistic)

A confidence interval (CI) gives a range within which the true mean μ is expected to lie. When σ is known (or estimated from a large pooled dataset), we use the z-statistic.

z-CI vs t-CI — which is wider?

The t-based CI is always wider than the z-based CI at the same confidence level and sample size, because t values are always larger than z (except at df = ∞ where t = z = 1.96). The t-CI is wider because it accounts for additional uncertainty from estimating σ from a small sample.

Worked example — Sulphate in river water (z vs t)

Three sets of 10 replicates give s₁ = 0.034, s₂ = 0.037, s₃ = 0.031 mg. Use these to find s_pooled, then construct (i) 95% CI for x̄ = 1.246 mg, N = 3 using z (σ = s_pooled), and (ii) 95% CI for a new set with x̄ = 1.198 mg, s = 0.042 mg, N = 3 using t.

s_pooled = √[(9×0.034² + 9×0.037² + 9×0.031²)/(30−3)] = 0.034 mg (used as σ)
(i) z = 1.96; CI = 1.246 ± 1.96×0.034/√3 = [1.207, 1.284] mg
(ii) df = 2, t = 4.30; CI = 1.198 ± 4.30×0.042/√3 = [1.094, 1.302] mg
CI (i) is narrower because z < t. CI (ii) wider due to t-statistic with low df (df = 2).

Comparing Two Means

Test whether two independently measured means are statistically significantly different at a given confidence level.

Two-means t-test — reject null if:
$$\lvert\bar{x}_1 - \bar{x}_2\rvert > t\cdot s_\text{pooled}\cdot\sqrt{\dfrac{N_1+N_2}{N_1 N_2}}$$
dfN₁ + N₂ − 2
s_pooledcalculated from the two datasets

If |difference| > right-hand side → reject null → means are significantly different.

Worked example — Chloride in concrete (two environments)

Env. A (full immersion, N=5): x̄_A = 15.30 mL, s_A = 0.158. Env. B (splash zone, N=5): x̄_B = 14.02 mL, s_B = 0.152. 95% CI. Is the difference significant?

s_pooled = √[(4×0.158² + 4×0.152²)/8] = 0.155 mL
df = 8, t = 2.31; threshold = 2.31 × 0.155 × √(10/25) = 0.227 mL
|x̄_A − x̄_B| = |15.30 − 14.02| = 1.28 mL
1.28 mL > 0.227 mL → difference IS statistically significant. Environment A (full immersion) has higher chloride → greater corrosion risk.

Detecting Systematic Error (Bias)

Bias test — reject null (no bias) if:
$$\lvert\bar{x} - x_t\rvert > \dfrac{t\cdot s}{\sqrt{N}}$$
xₜknown true value (e.g. certified reference material)
dfN − 1
Worked example — Soil contamination

Permissible Pb limit = 20 ppm. N = 16 measurements: x̄ = 19.36 ppm, s = 0.920 ppm. Is the soil contaminated beyond the limit?

df = 15, t = 2.13; threshold = 2.13 × 0.920/√16 = 0.49 ppm
|19.36 − 20| = 0.64 ppm
0.64 > 0.49 → reject null → soil IS statistically confirmed to contain Pb above the permissible limit.

Quick recap

Gravimetric Analysis

If you can convert your analyte into a pure solid of known formula, weigh it, and do some molar mass arithmetic — that's gravimetric analysis. It's one of the most accurate methods in analytical chemistry.

Four Types of Gravimetric Analysis

1. Precipitation

Analyte converted to insoluble precipitate, filtered, dried or ignited, weighed. Most common. Basis of this topic.

2. Volatilisation

Mass change from a volatile component (CO₂, H₂O) released by heating. Measured directly (mass loss) or by trapping in a weighed absorbent.

3. Electrogravimetry

Analyte electrodeposited onto a pre-weighed platinum electrode. Mass gain = mass of analyte. Very accurate — no coprecipitation.

4. Thermogravimetric Analysis (TGA)

Mass recorded continuously as temperature increases at a controlled rate. Decomposition steps appear as discrete mass-loss events.

Digestion — why it matters

After precipitation, heating the precipitate in the mother liquor (digestion) improves crystal quality via Ostwald ripening — small crystals dissolve and redeposit on larger ones. Result: fewer, larger crystals that filter more cleanly with less occlusion.

Gravimetric Factor and Calculations

Gravimetric factor (GF)
$$\text{GF} = \dfrac{a\cdot M_\text{analyte}}{b\cdot M_\text{precipitate}}$$
amoles of analyte per formula unit
bmoles of precipitate produced per formula unit

mass(analyte) = mass(precipitate) × GF

The GF converts directly from what you weigh (precipitate) to what you want to know (analyte).

Worked example — Impure pyrite → BaSO₄

An impure pyrite sample contains 90% FeS₂. How many grams must be dissolved to produce at least 1.00 g BaSO₄? (1 FeS₂ → 2 BaSO₄; MM(FeS₂) = 119.98; MM(BaSO₄) = 233.39)

n(BaSO₄) = 1.00/233.39 = 4.28×10⁻³ mol
n(FeS₂) = 4.28×10⁻³/2 = 2.14×10⁻³ mol (ratio 1:2)
m(pure FeS₂) = 2.14×10⁻³ × 119.98 = 0.257 g
m(impure pyrite) = 0.257 / 0.90 = 0.286 g
Worked example — Ferrocene: %Fe, then empirical formula

0.4873 g sample → 0.2091 g Fe₂O₃. Combustion of 0.5123 g → 1.2119 g CO₂ + 0.2482 g H₂O. Find %Fe, %C, %H, and identify compound.

n(Fe₂O₃) = 0.2091/159.69 = 1.309×10⁻³ mol → n(Fe) = 2.619×10⁻³ mol → m(Fe) = 0.1463 g
%Fe = 0.1463/0.4873 × 100 = 30.02%
n(CO₂) = 1.2119/44.01 = 0.02754 mol → n(C) = 0.02754 mol → m(C) = 0.3305 g → %C = 64.52%
n(H₂O) = 0.2482/18.02 = 0.01377 mol → n(H) = 0.02754 mol → m(H) = 0.02775 g → %H = 5.42%
Molar ratios: Fe : C : H = (30.02/55.85) : (64.52/12.01) : (5.42/1.008) = 0.5376 : 5.373 : 5.377
Divide by 0.5376 → Fe:C:H = 1:10:10 → Empirical formula FeC₁₀H₁₀ = ferrocene, Fe(C₅H₅)₂

Coprecipitation Errors

Surface adsorption

Ions adsorb onto crystal surface. Minimised by washing with appropriate wash solution.

Occlusion

Impurities physically trapped inside crystal during rapid growth. Cannot be removed by washing. Minimised by slow precipitation and digestion.

Mechanical entrapment

Mother liquor trapped within precipitate mass. Minimised by proper filtration.

Post-precipitation

A second compound nucleates on the precipitate surface during prolonged standing. Minimised by filtering promptly.

Volatilisation Gravimetry & TGA

TGA: sample heated at a controlled rate (~10°C/min); balance records mass continuously. Each decomposition reaction appears as a distinct mass-loss step. The derivative (DTG) shows a peak at each event.

Worked example — CaCO₃ in concrete by TGA

A 15.0293 mg sample loses 0.2201 mg between 600–800°C (CaCO₃ → CaO + CO₂). Is the CaCO₃ content below the 5 w/w% safe limit?

n(CO₂) = 0.2201×10⁻³ / 44.01 = 5.001×10⁻⁶ mol = n(CaCO₃)
m(CaCO₃) = 5.001×10⁻⁶ × 100.09 × 10³ mg/g = 0.5006 mg
w/w% = 0.5006/15.0293 × 100 = 3.33% → below 5% threshold ✓

Combustion Analysis

Organic sample burned completely in excess O₂. Products trapped in sequence: H₂O in Drierite (CaSO₄), then CO₂ in Ascarite (NaOH on silica). Mass gain of each trap gives moles of H and C.

  • n(H) = 2 × m(H₂O)/18.015
  • n(C) = m(CO₂)/44.009
  • %N determined separately by Kjeldahl method or elemental analyser

Properties of Electromagnetic Radiation

Wave equation & derived quantities
$$c = \lambda\nu \qquad \bar{\nu} = \dfrac{1}{\lambda} \qquad E = h\nu = \dfrac{hc}{\lambda} = hc\bar{\nu}$$
cspeed of light = 2.998×10⁸ m s⁻¹ (in vacuum)
λwavelength (m, nm, or cm)
υfrequency (Hz = s⁻¹)
ν̄wavenumber (cm⁻¹); ν̄ = 10⁷/λ(nm)
hPlanck's constant = 6.626×10⁻³⁴ J s
Ephoton energy (J) — proportional to frequency
Radiation intensity
$$I = \dfrac{P}{A} = \dfrac{E\Phi}{A}$$
Ppower (W)
Across-sectional area (m² or μm²)
Φphoton flux (photons s⁻¹)
When light enters a medium

Velocity v and wavelength λ change. Frequency υ and energy E remain constant. The wavenumber and energy travel with the wave irrespective of the medium.

Worked example — Hα Balmer line (λ = 656 nm)

Find frequency, wavenumber, and intensity of a beam with Φ = 5×10¹⁴ photons/s through A = 3.15 μm².

υ = c/λ = 3×10⁸/(656×10⁻⁹) = 4.57×10¹⁴ Hz
ν̄ = 10⁷/656 = 15,244 cm⁻¹
E = hc/λ = 6.626×10⁻³⁴ × 3×10⁸/(656×10⁻⁹) = 3.03×10⁻¹⁹ J
I = 3.03×10⁻¹⁹ × 5×10¹⁴ / 3.15 = 4.81×10⁻⁵ W μm⁻²

Atomic Energy Transitions

Absorption

Photon absorbed → atom goes from ground to excited state. Requires E_photon = ΔE exactly (quantised).

Emission

Excited atom releases photon → returns to lower state. Emitted λ characteristic of the element.

Fluorescence

Rapid radiative emission after absorption. Spin-allowed → lifetime ~ns. Same or longer λ than absorbed.

Phosphorescence

Delayed emission. Spin-forbidden transition → lifetime ms to s.

Non-radiative relaxation

Energy dissipated as heat. Competes with fluorescence/phosphorescence. Lowers fluorescence quantum yield.

Beer-Lambert Law

Transmittance, Absorbance, Beer-Lambert
$$T = \dfrac{I}{I_0} \qquad A = -\log T = \log\dfrac{I_0}{I} = \varepsilon b c$$
Ttransmittance (0–1); %T = T×100
Aabsorbance — dimensionless, additive for multiple species
εmolar absorptivity (L mol⁻¹ cm⁻¹) — maximum at λ_max
bpath length (cm) — usually 1 cm cuvette
cconcentration (mol L⁻¹)

Valid only for: monochromatic radiation, dilute solutions, no fluorescence, no chemical equilibria changing with concentration.

Worked example — β-carotene (λ = 445 nm)

Φ₀ = 4.75×10¹⁵, Φ = 2.81×10¹⁵ photons/s; A = 3.15 μm². Find T and A.

E = hc/λ = 6.626×10⁻³⁴ × 3×10⁸/(445×10⁻⁹) = 4.47×10⁻¹⁹ J
I₀ = 4.47×10⁻¹⁹ × 4.75×10¹⁵/3.15 = 6.74×10⁻⁴ W μm⁻²; I = 3.99×10⁻⁴ W μm⁻²
T = 3.99/6.74 = 0.592; A = −log(0.592) = 0.228

Blank Correction

A blank (solvent + cuvette, no analyte) is measured to remove background absorbance from the solvent, cuvette, and instrument optics.

Blank-corrected absorbance
$$A_\text{analyte} = -\log\dfrac{I_A}{I_B}$$
I_Aintensity transmitted through analyte solution
I_Bintensity transmitted through blank

If raw I₀ is used instead of I_B (blank), the measured absorbance will be overestimated — the solvent's own absorption is incorrectly attributed to the analyte.

Four Methods of Determining Concentration

Method A — Known ε

C = A / (ε·b)

Quick. Requires accurate prior knowledge of ε for the analyte at the measurement wavelength and conditions.

Method B/C — Direct comparison

C_analyte = (A_analyte / A_std) × C_std

ε and b cancel. Requires standard prepared under identical conditions to sample. Simple and robust.

Method D — Standard addition

C_analyte = [A_analyte / (A_spiked − A_analyte)] × C_spike

Add known spike of standard to the unknown sample. Compensates for matrix effects — any other species affecting absorbance affects both measurements equally and cancels.

Worked example — BP-3 sunscreen (known ε)

λ = 320 nm; I₀ = 4.342 W m⁻², I = 1.206 W m⁻²; ε = 13,028 L mol⁻¹ cm⁻¹; b = 1 cm; sample volume = 250 mL; sample mass = 53.01 mg. Find %w/w BP-3.

A = −log(1.206/4.342) = 0.556
C = 0.556/(13028 × 1) = 4.27×10⁻⁵ mol/L
n(BP-3) = 4.27×10⁻⁵ × 0.250 = 1.07×10⁻⁵ mol; m(BP-3) = 1.07×10⁻⁵ × 228.24 = 2.44 mg
%w/w = 2.44/53.01 × 100 = 4.60%
Worked example — Indium in InP semiconductor (standard addition)

A_sample = 1.002; add 5 mL of 2.50×10⁻³ M In standard to 100 mL solution → A_spiked = 1.102. Find C(In) in sample.

C_spike = 2.50×10⁻³ × 5/100 = 1.25×10⁻⁴ M (concentration added to solution)
C_sample = [1.002 / (1.102 − 1.002)] × 1.25×10⁻⁴ = [1.002/0.100] × 1.25×10⁻⁴
C_sample = 1.25×10⁻³ M

UV-Vis Spectrometer Components

Signal path: Source → Wavelength selector → Sample → Detector → Signal processor

Sources

Deuterium lamp: UV 150–400 nm, continuous spectrum.
Tungsten-halogen: Vis 400–800 nm, ~3500 K blackbody.

Wavelength selectors

Diffraction grating: nλ = d sinθ; tunable by rotation.
Prism: Snell's law refraction; quartz for UV+Vis, glass for Vis only.

Cuvettes

Quartz: transparent 160–3000 nm → use for UV.
Glass/plastic: Vis only → never use for UV measurements.

Detectors

PMT: cascade dynode amplification, ~150–1000 nm, very sensitive.
Photodiode: solid state, 350–1100 nm, fast; used in diode-array instruments.

Single-beam vs double-beam

Single-beam: blank and sample measured sequentially through the same path — susceptible to source drift between measurements.
Double-beam: beam split to reference and sample paths simultaneously — compensates source fluctuations, more accurate.

Fundamentals

Key terms
  • Titrant: known-concentration reagent added from the burette
  • Equivalence point (EQ): stoichiometric amount of titrant added — theoretical, not directly observable
  • End point (EP): experimentally observed signal change (colour, pH, conductivity)

Strong Acid / Strong Base Titrations

pH at equivalence point = 7 (neutral salt formed, 25°C).

Choosing an indicator

The indicator's colour-change range must overlap the equivalence point pH. For strong acid/strong base (EQ at pH 7), any indicator with range between pH ~4–10 is acceptable due to the steep titration curve near EQ.

Weak Acid / Strong Base

When the acid is weak, it doesn't fully dissociate. This changes the shape of the titration curve and moves the equivalence point above pH 7.

Weak acid titration curve formulas (HA + NaOH)
$$\text{Start:}\quad \text{pH} = -\log\sqrt{K_a C_\text{HA}}$$
$$\text{Buffer:}\quad \text{pH} = \text{p}K_a + \log\dfrac{[\text{A}^-]}{[\text{HA}]}$$
$$\text{Half-EQ:}\quad \text{pH} = \text{p}K_a \quad (\text{when }[\text{A}^-] = [\text{HA}])$$
$$\text{EQ point:}\quad \text{pH} = -\log\sqrt{\dfrac{K_w K_a}{C_\text{AB}}} \;\longrightarrow\; \text{pH} > 7$$
⚠️ Equivalence point for weak acid/strong base is BASIC

At the equivalence point, the analyte has been fully converted to its conjugate base A⁻. This base is weakly basic, so the solution is pH > 7. Choose an indicator whose range is above 7 — e.g. phenolphthalein (pH 8.5–10.5).

Argentometric & Volhard Method

Solubility product
$$K_{sp} = [\text{A}]^a[\text{B}]^b \quad (\text{for } A_aB_b \rightleftharpoons a\text{A} + b\text{B})$$

AgCl: Ksp = 1.8×10⁻¹⁰ · AgBr: 5.3×10⁻¹³ · AgI: 8.5×10⁻¹⁷ · AgSCN: 1.0×10⁻¹²

Volhard method — back-titration for halides
  1. Add known excess AgNO₃ to acidified halide sample → precipitate AgX
  2. Filter off AgX (critical step — must remove before back-titrating)
  3. Add Fe(NO₃)₃ indicator to filtrate
  4. Back-titrate excess Ag⁺ with standard KSCN → AgSCN precipitates
  5. Endpoint: Fe³⁺ + SCN⁻ → FeSCN²⁺ (first permanent red colour)

n(X⁻) = n(Ag total) − n(SCN) = C_AgNO₃·V_AgNO₃ − C_KSCN·V_KSCN

Worked example — Volhard for Br⁻

25.00 mL sample + 25.00 mL 0.0123 M AgNO₃. Excess Ag⁺ back-titrated with 7.45 mL 0.0077 M KSCN. Find [Br⁻].

n(SCN) = 0.0077 × 0.00745 = 5.74×10⁻⁵ mol = n(Ag excess)
n(Br⁻) = 3.075×10⁻⁴ − 5.74×10⁻⁵ = 2.50×10⁻⁴ mol
C(Br⁻) = 2.50×10⁻⁴/0.02500 = 0.0100 M

Redox Titrations

Reaction spontaneity from standard potentials
$$E^\circ_\text{rxn} = E^\circ_\text{red} - E^\circ_\text{ox} > 0 \implies \text{spontaneous}$$

More positive E⁰ → species more easily reduced (stronger oxidising agent). Reaction: higher E⁰ species is reduced; lower E⁰ species is oxidised.

KMnO₄ / Fe²⁺ (ratio 1:5)

E⁰_rxn = 1.51 − 0.77 = +0.74 V ✓

Self-indicating: endpoint = first permanent purple colour (excess MnO₄⁻ not reduced)

Run in dilute H₂SO₄ (not HCl — Cl⁻ interferes)

MCQ Traps

Plausible-but-wrong claims the exam uses. Read each one. Know what's actually true.

Topics 1 & 2 — Statistics

Stats T1
"Standard deviation measures accuracy."
FALSE. Standard deviation measures precision. Accuracy is measured by error (how close to the true value). You can have high precision with poor accuracy.
Stats T2
"The z-based confidence interval is wider than the t-based CI at the same confidence level."
FALSE. The t-based CI is always wider. t accounts for additional uncertainty from estimating σ from a small sample. t-values are always ≥ z (equal only at df = ∞).
Stats T3
"In the Q-test, the gap is the distance from the suspect value to the next-nearest neighbour."
FALSE. The gap is |x_q − nearest neighbour|. Always sort the data first, then take the difference to the adjacent value.
Stats T4
"After identifying an outlier with the Q-test, you can include it in the recalculated mean."
FALSE. After rejecting an outlier, recalculate mean and SD on the remaining N−1 data points only.
Stats T5
"The sample standard deviation formula divides by N."
FALSE. Sample SD divides by N−1 (Bessel's correction) to give an unbiased estimate of σ. Only the population SD divides by N.

Topic 3 — Gravimetric

Grav T1
"1 mole of FeS₂ produces 1 mole of BaSO₄."
FALSE. FeS₂ contains 2 sulfur atoms, each forming one SO₄²⁻ → one BaSO₄. So 1 mol FeS₂ → 2 mol BaSO₄. Forgetting this stoichiometry halves your answer.
Grav T2
"Occlusion can be removed by washing the precipitate."
FALSE. Occluded impurities are physically trapped inside the crystal lattice — washing cannot reach them. Only digestion (slow crystal growth, Ostwald ripening) reduces occlusion.
Grav T3
"TGA mass loss between 600–800°C represents the mass of CaO lost."
FALSE. The mass lost is CO₂ (from CaCO₃ → CaO + CO₂). CaO remains in the crucible. Use m(CO₂) to find n(CO₂) = n(CaCO₃), then calculate m(CaCO₃).
Grav T4
"Electrogravimetry requires a gravimetric factor calculation."
FALSE. In electrogravimetry, the analyte is deposited as the pure element on the electrode — mass gain directly equals mass of analyte. No GF needed.

Topic 4 — Spectroscopy

Spec T1
"When light enters a denser medium, its frequency decreases."
FALSE. Frequency (and energy) are unchanged when light changes media. Only velocity and wavelength change (both decrease in a denser medium).
Spec T2
"Measuring absorbance without a blank is fine because the blank has zero absorbance."
FALSE. Solvents, cuvettes, and dissolved species all absorb. Without blank correction you overestimate the analyte absorbance. Always measure I_blank.
Spec T3
"In standard addition: C_analyte = A_analyte / A_spiked × C_spike."
FALSE. The denominator is the difference: C_analyte = A_analyte / (A_spiked − A_analyte) × C_spike. Using A_spiked directly gives a systematically low result.
Spec T4
"Glass cuvettes can be used for UV measurements."
FALSE. Glass absorbs UV radiation. Use quartz (SiO₂) cuvettes for UV. Glass and plastic are only suitable for Vis (400–800 nm).

Topic 5 — Titrations

Tit T1
"The equivalence point for a weak acid/strong base titration is at pH 7."
FALSE. The equivalence point is basic (pH > 7) because the product is the conjugate base A⁻, which is weakly basic. Choose an indicator that changes above pH 7 (e.g. phenolphthalein).
Tit T2
"EDTA reacts with Ca²⁺ in a 2:1 molar ratio."
FALSE. EDTA always reacts 1:1 with any metal ion regardless of its charge. Ca²⁺, Fe³⁺, and Th⁴⁺ all consume exactly 1 mole of EDTA per mole of metal.
Tit T3
"In the Volhard method, you add KSCN directly to the original sample without filtering."
FALSE. You must filter off the AgX precipitate first. If AgCl remains, SCN⁻ reacts with it at the surface (Ksp AgSCN < Ksp AgCl) giving a falsely high Ag⁺ value and wrong Cl⁻ result.
Tit T4
"The molar ratio of MnO₄⁻ to Fe²⁺ in permanganate titration is 1:1."
FALSE. MnO₄⁻ accepts 5 electrons (Mn⁷⁺ → Mn²⁺); Fe²⁺ donates 1 electron. Ratio is 1 MnO₄⁻ : 5 Fe²⁺. Forgetting this gives a result that is 5× too low.
Tit T5
"KMnO₄ titrations require an external indicator."
FALSE. KMnO₄ is self-indicating. The purple MnO₄⁻ is immediately decolourised by Fe²⁺ before the endpoint. The first permanent purple colour marks the endpoint. No separate indicator is needed.
Tit T6
"Eriochrome Black T indicator can be used at pH 5."
FALSE. At pH < 6.3, free EBT is in the red H₂EBT⁻ form — the same colour as the metal-EBT complex. You cannot see the endpoint. EBT must be used at pH > 6.3 (typically pH 10).
Tit T7
"The half-equivalence point pH equals the pKb for a weak acid titration."
FALSE. At the half-equivalence point of a weak acid titration, pH = pKa. The pKb relates to the conjugate base. Don't confuse which ionisation constant applies.

Example 1 — Sulphate in river water (CI comparison)

3 sets of 10 replicates give s₁=0.034, s₂=0.037, s₃=0.031 mg. Part (i): x̄=1.246 mg, N=3, use z (σ=s_pooled). Part (ii): x̄=1.198 mg, s=0.042 mg, N=3, use t. Construct 95% CIs and explain which is wider.

(ii) df=2, t=4.30: CI = 1.198 ± 4.30×0.042/√3 = [1.094, 1.302] mg

Large discrepancy → the vitamin tablet matrix contains other absorbing species. Standard addition is more accurate because it compensates for matrix effects — Method B overestimates or underestimates depending on whether matrix suppresses or enhances absorbance.

Q1.2 — Errors
An object has true diameter x_t = 10.0 cm. Three measurements: 9.8, 10.2, 9.9 cm.

  • (b) Average absolute error
  • (c) Relative errors R₁, R₂, R₃ and average relative error

Q1.3 — Gaussian
30 industrial process yield measurements. Mean = 91.56%. Sorted median values: 91.5, 91.6%.
  • (a) Calculate the mean and median
  • (b) Why might you prefer one over the other?
Mean = 91.56% (given); Median = (91.5+91.6)/2 = 91.55%
Mean is sensitive to outliers; median is more robust. For a near-symmetric distribution with no outliers, both are similar. The mean uses all data and is the best estimator when the distribution is Gaussian.
Q2.1 — Two means
Two aspirin tablet batches (N=5 each). Batch 1: x̄=250.8 mg, s=7.312 mg. Batch 2: x̄=248.0 mg, s=8.454 mg. At 95% confidence, are the batches statistically different?
s_pooled = √[(4×7.312²+4×8.454²)/8] = 7.904 mg/tablet; df=8, t=2.31
Threshold = 2.31×7.904×√(10/25) = 11.55 mg/tablet
|250.8−248.0| = 2.8 mg/tablet
2.8 < 11.55 → NO statistically significant difference. Batches are equivalent at 95% CL.
Q2.2 — Two means
Polymer injectability issue. Lot X (N=4): x̄=151,500 g/mol. Lot Y (N=7): x̄=156,000 g/mol. s_pooled=3,472 g/mol. df=9, t=2.26. Is the molecular weight difference responsible for the injectability problem?

Q2.3 — Q-test
Pump flow rates: 112, 115, 110, 113, 116, 114, 103, 115, 119, 112 m³/min. Is 103 an outlier at 90% confidence? If rejected, recalculate mean and SD.
Sorted: 103, 110, 112, 112, 113, 114, 115, 115, 116, 119
Q = |103−110|/(119−103) = 7/16 = 0.437; Q_crit(N=10, 90%) = 0.412
0.437 > 0.412 → outlier, reject 103
Recalculate (N=9): x̄ = (110+112+112+113+114+115+115+116+119)/9 = 114.0 m³/min, s = 2.8 m³/min
Q2.4 — Bias test
Titanium content of standard reference material (certified = 15.66%). N=4 measurements: x̄=15.64%, s=0.315%. Is there a systematic error at 95% confidence?
df=3, t=3.18; threshold = 3.18×0.315/√4 = 0.501%
|15.64−15.66| = 0.02%
0.02 < 0.501 → null hypothesis NOT rejected. No statistically significant systematic error (bias) in this method.
Q2.5 — Bias test
Lead in soil: permissible limit = 20 ppm. N=16 measurements: x̄=19.36 ppm, s=0.920 ppm. Is the soil contaminated beyond the permissible limit at 95% confidence?
df=15, t=2.13; threshold = 2.13×0.920/√16 = 0.490 ppm
|19.36−20| = 0.64 ppm
0.64 > 0.49 → reject null → soil IS statistically confirmed to contain Pb above the permissible limit. Remediation required.
